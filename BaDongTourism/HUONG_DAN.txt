----- BƯỚC 1: TẠO DATABASE -----

1. Mở SQL Server Management Studio (SSMS)
2. Kết nối đến SQL Server của bạn
3. Mở file: Database\BaDongTourism.sql
4. Nhấn F5 để chạy toàn bộ script
   → Script sẽ tự tạo database "BaDongTourism" cùng tất cả bảng và dữ liệu mẫu

----- BƯỚC 2: CẤU HÌNH CONNECTION STRING -----

Mở file Web.config, tìm dòng:

  <add name="BaDongDB"
       connectionString="Data Source=.\SQLEXPRESS;Initial Catalog=BaDongTourism;Integrated Security=True"
       ... />

Chỉnh sửa "Data Source" theo cài đặt SQL Server của bạn:
  - SQL Server Express local  : Data Source=.\SQLEXPRESS
  - SQL Server Developer local: Data Source=(local)  hoặc  Data Source=.
  - SQL Server có tên         : Data Source=TEN_MAY\SQLEXPRESS
  - SQL Server Authentication : thêm  User ID=sa;Password=matkhau;

----- BƯỚC 3: MỞ PROJECT TRONG VISUAL STUDIO 2022 -----

1. Mở Visual Studio 2022
2. File → Open → Project/Solution
3. Chọn file: BaDongTourism.sln
4. Chờ VS load project

----- BƯỚC 4: BUILD & CHẠY -----

1. Nhấn Ctrl+Shift+B để Build
2. Nhấn F5 (hoặc Ctrl+F5) để chạy
3. Trình duyệt sẽ mở tự động tại: http://localhost:PORT/

----- ĐĂNG NHẬP ADMIN -----

URL    : http://localhost:PORT/Admin/Login.aspx
User   : admin
Pass   : admin123

→ Sau khi đăng nhập: Dashboard → quản lý toàn bộ nội dung

----- LƯU Ý QUAN TRỌNG -----

1. Tạo thư mục Content/images/uploads/ (để upload ảnh hoạt động)
2. Đảm bảo IIS Express hoặc IIS có quyền ghi vào thư mục uploads/
3. Nếu dùng SQL Server Authentication, đổi Integrated Security=False
   và thêm User ID=...; Password=...;
4. Đổi mật khẩu admin sau khi deploy lên production
